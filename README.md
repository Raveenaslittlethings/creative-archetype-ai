# Creative Archetype AI
An AI-powered self-discovery tool that identifies your dominant creative personality archetype and provides personalized growth insights.

## Vision
This project blends psychology, creativity, and AI to help individuals:
- Understand their creative tendencies
- Identify strengths and blind spots
- Receive growth-oriented guidance
- Reflect deeply on their personal development

## The 5 Archetypes
- Visionary – The future thinker
- Creator – The expressive builder
- Analyst – The logical architect
- Explorer – The curious experimenter
- Strategist – The intentional planner

## Features
- Interactive personality assessment
- Archetype scoring engine
- AI-generated personalized insights
- Memory layer for tracking previous sessions
- Web-based interface (Streamlit)

## Tech Stack
- Python
- Streamlit
- OpenAI API
- JSON for memory storage

##  Future Improvements
- Emotional tone detection
- Career recommendation engine
- Archetype blending (multi-type personality)
- Data visualization dashboard
